Snipe Server API Documentation

package snipe.lib

class Temp

Local temp directory access module.

Methods

load (key:String):Dynamic

Loads and parses a file from local temp directory containing a string in JSON format.

Parameters:

key

File name.

Returns:

Anonymous object. Returns null if the file does not exist or is empty.

save (key:String, data:Dynamic, ?time:Int):Void

Saves data to a file in local temp directory serialized into JSON format. When time parameter is specified, file will be written to disk only if this amount of seconds passed since last save.

Parameters:

key

File name.

data

Anonymous object with data.

time

Checks if this amount of seconds has passed since last write.

Snipe Server API Documentation

package snipe.lib

data
AtomicInt

Atomic integer variable.

AtomicIntCyclic

Cyclic atomic integer variable.

Config

Configuration file API - loading, parsing and access to variables.

LazyIntMap

Key-value map with Int keys and elements initialized on first get() call.

LazyMap

Key-value map with String keys and elements initialized on first get() call.

LocalStats

Local stats counters class (for usage in cache server or slave server modules). Counts stats events by ID and resets daily.

LocaleMap

Localization objects map for a single database table row ID.

Logger

Generic logging class that runs in a separate thread.

MacroClient

Client instance macros.

MacroUser

User instance macros.

Params

Client request parameters. Used in both slave server <-> cache server and game client <-> slave server communication.

ParserFODS

FODS (OpenDocument Flat XML Spreadsheet) file format parser. Will parse a FODS file containing static data tables into a list of arrays.

QueueProducerSafe

Lock-free queue for multiple producers and one consumer.

QueueSafe

Lock-free queue for multiple producers and multiple consumers.

QueueUnsafe

Lock-free queue for one producer and one consumer.

RealtimeStats

Real-time stats collection and logging. Needs to be extended with stats container type given.

Temp

Local temp directory access module.

Util

Miscellaneous utility functions.

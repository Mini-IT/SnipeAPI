class UserLogicTreeCore

User logic tree state class. Logic is described in detail in Nodes article.

Fields

Tree ID.

read only info:LogicTreeCore

Tree info shortcut.

read only node:UserLogicNodeCore

Current tree node state.

Methods

dump ():Dynamic

Stores user logic tree state into an anonymous object for client response.

Returns:

Anonymous object with user logic tree parameters.

enter (nodeInfo:LogicNodeCore):UserLogicNodeCore

Enters a given node. This is a low-level function that does not do any checks for node requirements. In most cases UserLogicCore.enterNext() is preferable.

Parameters:

Node info.

Returns:

New user node instance.

class CacheModule

extends ModuleCache<CacheServer>

Leaderboards package cache server module.

ALPHA: This module and all underlying classes are currently in alpha.

Include in cache server with:

    loadModules([
      snipe.packages.leaderboard.CacheModule,
      ]);

Methods

clear (id:String):Void

Clears the given leaderboard immediately.

Parameters:

id

Leaderboard string ID.

get (id:String):Array<Dynamic>

Returns leaderboard contents with this string ID.

Parameters:

id

Leaderboard string ID.

Returns:

An array of leaderboard records. Returns null if the leaderboard is not found.

getYesterday (id:String):Array<Dynamic>

Returns yesterday's leaderboard contents with this string ID.

Parameters:

id

Leaderboard string ID.

Returns:

An array of leaderboard records. Returns null if the leaderboard is not found or does not support saving yesterday's contents.

has (id:String):Bool

Returns true if there is a leaderboard with this ID.

Parameters:

id

Leaderboard string ID.

Returns:

True if the leaderboard exists. False otherwise.

init (opts:_LeaderboardOptions):Void

Initializes and loads a leaderboard with this ID. If the leaderboard already exists, nothing happens.

Parameters:

opts

Leaderboard options.

update (id:String, row:Dynamic):Bool

Updates the given leaderboard with the given record if necessary.

Parameters:

id

Leaderboard string ID.

row

Full leaderboard record. Has to have "id" and "points" field and can contain any other fields that will be stored in leaderboard for later use. All fields must be serializable and deserializable into JSON.

Returns:

True if this record is now in the leaderboard. False otherwise, including when the record was already there.

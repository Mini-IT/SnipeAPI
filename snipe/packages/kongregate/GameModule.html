package snipe.packages.kongregate

class GameModule

extends Module<ClientGame, ServerGame>

Kongregate login package game server module.

Client should send "kongregate_game_auth_token" field in "user.login" request.If it is not received, "user.login" response will return "noKongregateAuthToken" error code.

In case when Kongregate login fails (for example, wrong auth token or wrong API key), the client will receive a "packages/kongregate.login" message with "loginFailure" error code. Client will not be disconnected from the server.

Include in game server with:

    loadModules([
      snipe.packages.kongregate.GameModule,
      ]);

Game server configuration variables:

  • sns.KGapiKey - Kongregate project API key

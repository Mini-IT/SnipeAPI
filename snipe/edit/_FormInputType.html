package snipe.edit

enum _FormInputType

Editor form input type enum.

Values

INPUT_TEXT

Default text field.

INPUT_PASSWORD

Password text field.

INPUT_FILE

File.

INPUT_VALUE

Uneditable value.

INPUT_TEXTAREA

Text area.

INPUT_SELECT

Drowdown list of values.

INPUT_HEADER

Group header.

INPUT_JSON

JSON-string.

INPUT_CHECKBOX

Checkbox.

INPUT_DATETIME

Date/Time string (DD.MM.YYYY HH:MM).

INPUT_HIDDEN

Hidden field with a given value.

Snipe Server API Documentation

package snipe.edit.modules

typedef _CalendarEvent

Calendar event type. Calendar events API is described in detail in Calendar Events article.

Properties

optional inputs:Array<_BlockFormInput>

Additional inputs for add/edit event form.

name:String

Event name.

optional noEndDate:Bool

If enabled, event is active indefinitely.

optional params:Array<String>

Add/update event parameters (supports <type>:<name> naming scheme).

optional stageInputs:Array<Array<_BlockFormInput>>

Form inputs for event stage parameters.

optional stageParams:Array<Array<String>>

Event stage parameters.

type:String

Event type.

optional editPost (:Int, :Dynamic, :Vars):Void

Function to output additional page blocks after main form on edit event page.

Function arguments are: event ID, event parameters, request variables.

optional listField (:Int, :Dynamic):Void

Function to output event data in event list.

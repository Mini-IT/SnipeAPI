Snipe Server API Documentation

package snipe.slave.modules

interface _HookUserDailyLoginPost

implemented by UserModuleCore

User module events interface. Users are described in detail in Users article. Module subscription API is described in detail in Subscriptions article.

Methods

userDailyLoginPost (user:UserClass, lastDay:Int, currentDay:Int):Void

Event hook. Called for each user after first successful login of the day if the module is subscribed to it. You can subscribe the module to this event in the constructor like this:

server.subscribeModule("core/user.dailyLoginPost", this);

Note: If more than a year has passed since previous login, you will not be able to determine that in this hook.

Note: The new day begins at 00:00 of server time.

Parameters:

user

User instance.

lastDay

Last day of the year the user logged in (1-366).

currentDay

Current day of the year (1-366).

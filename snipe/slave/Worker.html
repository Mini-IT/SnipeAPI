Snipe Server API Documentation

package snipe.slave

class Worker

Asynchronous worker thread. Can be created in any server module.

Constructor

new (s:Server, tn:String)

Parameters:

s

Server variable.

tn

Short thread name. The full thread name will be async: <name>.

Methods

call (name:String, params:Dynamic, workerFunc:Dynamic ‑> Void, responseFunc:Dynamic ‑> Void):Void

Adds asynchronous call to the queue.

Parameters:

name

Call name.

params

Call parameters. Will be given to worker method and later to response method. You can modify them in worker method.

workerFunc

Worker method. Called in worker thread and accepts call parameters as an argument.

responseFunc

Response handler. Called in main server thread and accepts call parameters as an argument.
